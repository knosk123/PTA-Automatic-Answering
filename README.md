# PTA自动答题
参考项目 https://github.com/lmengx/fuckpta.git

感谢原项目作者和相关开源生态。

## 改进与优化
1.解决了中转站api无法调用问题。

2.解决了判断题无法作答的问题。

3.解决了部分填空题无法作答的问题。
